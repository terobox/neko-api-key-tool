# 写在前面
```
docker build -t your-image-name .
docker run -p 8000:80 your-image-name
```

# 使用方法
复制.env.example文件为.env
```
cp .env.example .env
```
修改.env文件中的配置
```
# 展示使用明细
SHOW_DETAIL="false"

# 展示余额
SHOW_BALANCE="true"

# BaseURL 结尾不要带/
BASE_URL="https://nekoapi.com"
```

展示使用明细功能需要使用的魔改版One API，项目地址：https://github.com/Calcium-Ion/one-api
